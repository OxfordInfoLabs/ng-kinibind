Action

Bind

Simple AJAX request binding to model in the view.

Example

In your component.ts create a reference to KinibindModel instance:
public bindModel: KinibindModel = new KinibindModel();

Pass this into the [model] property
<kb-bind [model]="bindModel">

Set your AJAX URL and method type (defaults to GET)
<kb-bind source="https://jsonplaceholder.typicode.com/users" method="GET" [model]="bindModel"></kb-bind>

The results of the AJAX request will be passed into bindModel. Result can be accessed on the model as follows:
Array bindModel.results
JSON bindModel.item
String||Integer bindModel.value

Results

{{user.name}}
{{user.email}} {{user.website}}
